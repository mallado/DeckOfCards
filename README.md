## Getting Started
### Prerequisites
You must have the following software installed 
* Java 16
* Maven 3.8.1 or later 
* GitHub console
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mallado/DeckOfCards.git
   ```
2. Compile code
   ```sh
   mvn clean install
   ```
